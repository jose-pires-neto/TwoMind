## 🚀 Instalação Rápida

### Método 1: Instalador Automático (Recomendado)

```bash
# Clone ou baixe os arquivos

```

O instalador automático:
- ✅ Verifica compatibilidade do sistema
- ✅ Instala todas as dependências
- ✅ Baixa modelos de IA otimizados
- ✅ Cria atalhos na área de trabalho
- ✅ Configura ambiente completo

### Método 2: Instalação Manual

```bash
# 1. Instale as dependências
pip install -r requirements_enhanced.txt

# 2. Execute a aplicação
python app_enhanced.py
```

## 🎮 Como Usar

### 🔐 Primeira Execução

1. **Configure Senha**: Na primeira vez, defina uma senha forte
2. **Aguarde Túnel**: O sistema criará automaticamente um túnel Cloudflare
3. **Acesse**: Use a URL fornecida para acessar de qualquer lugar

### 📝 Gerenciando Memórias

#### Adicionar Nova Memória
- **Interface**: Clique no botão `+` ou use `Ctrl+N`
- **Categorização**: O sistema categoriza automaticamente
- **Preview**: Veja a categoria sugerida antes de salvar

#### Editar Memórias
- **Clique**: Toque em qualquer memória no grafo
- **Editar**: Use o botão "Editar" no modal
- **Recategorização**: Sistema atualiza categoria automaticamente

#### Deletar Memórias
- **Seleção**: Abra qualquer memória
- **Confirmação**: Use o botão "Deletar" com confirmação
- **Limpeza**: Remove conexões associadas automaticamente

### 🔍 Sistema de Busca

#### Busca Básica
```
Digite: "trabalho projeto python"
Resultado: Encontra memórias relacionadas a projetos de trabalho em Python
```

#### Busca com IA
```
Pergunta: "Como estava meu humor na semana passada?"
IA: Analisa memórias e fornece resposta contextual inteligente
```

#### Filtros Avançados
- **Por Categoria**: Filtre apenas trabalho, pessoal, etc.
- **Por Data**: Ordenação cronológica
- **Por Relevância**: Baseado em similaridade semântica
- **Por Alfabética**: Ordem A-Z

### 📊 Visualização

#### Camadas Neurais
- **Ativação**: Botão de camadas no canto superior direito
- **Organização**: Memórias organizadas em anéis concêntricos
- **Densidade**: Mostra densidade de conexões

#### Filtros Visuais
- **Categorias**: Clique nos filtros coloridos no topo
- **Cores**: Cada categoria tem cor única
- **Conexões**: Links mostram relacionamentos

### ⌨️ Atalhos de Teclado

| Atalho | Ação |
|--------|------|
| `Ctrl+N` | Nova memória |
| `Ctrl+F` | Buscar |
| `Ctrl+R` | Centralizar vista |
| `Enter` | Busca normal |
| `Ctrl+Enter` | Busca com IA |
| `Esc` | Fechar modais |

## 🐛 Solução de Problemas

### Problemas Comuns

#### "IA não responde"
```bash
# Reinstale dependências de IA
pip install --upgrade transformers torch sentence-transformers
```

#### "Conexão recusada"
```bash
# Verifique se o servidor está rodando
python app_enhanced.py
# Aguarde mensagem de "Servidor iniciado"
```

#### "Memórias não aparecem"
```bash
# Verifique arquivos de dados
ls -la *.faiss *.pkl
# Se ausentes, serão criados automaticamente
```

#### "Categorização incorreta"
- Sistema aprende com uso
- Edite memórias para melhorar
- Aguarde mais dados para precisão

### Logs e Debug

**Ativação**:
```python
# No app_enhanced.py, linha ~50
app.run(host=host, port=port, debug=True)  # True para debug
```

**Localização de Logs**:
- Terminal onde rodou `python app_enhanced.py`
- Arquivo `tunnel_url.txt` para URL externa

### Backup e Restauração

**Backup Manual**:
```bash
# Copie estes arquivos importantes
cp my_index.faiss backup/
cp notes.pkl backup/
cp categories.pkl backup/
cp user_config.json backup/
```

**Restauração**:
```bash
# Cole os arquivos de volta
cp backup/*.faiss .
cp backup/*.pkl .
cp backup/*.json .
```
